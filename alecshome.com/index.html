What's this domain used for?

Because I already own the site AmazinAxel.com, why would I register and create another personal website? Well, having a seperate domain helps to keep things more secure, as I can have my developer site and restricted APIs hosted on AlecsHome.com rather than the AmazinAxel.com domain, which is more public. Additionally, I like to keep my sites organized and I have found that seperating the domains really helps. There are many other reasons for keeping this domain name, such as the fact that it includes my (actual) name in it which makes it much more personal and recognizable for emails and more personal content.

Are there any future plans for this domain and site?

As of now, I don't have any plans of moving any public-facing content (like my weekly SWR's) from my AmazinAxel domain to this one. I also will continue to keep my backend applications on here to keep things secure. This site will continue to gain more features (like dev testing for future projects) but won't overtake the AmazinAxel site on public-facing content.
